org.eclipse.mylyn.github by devsprint (forked from dgreen99)

Eclipse Mylyn to GitHub connector.

About

This project provides a Mylyn connector to the GitHub issue tracker. It was originally forked from the original version by Christian Trutz and others.

You can find out more about this fork from the Project Wiki

Installation

Install into Eclipse using the following update site: http://devsprint.github.com/org.eclipse.mylyn.github/site
